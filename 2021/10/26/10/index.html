    2. 创建映射文件

    安装PC端Docker

    PC端Docker下载 说明：这里是X64的安装包，非X64请自行前往官网下载对应版本。

    下载完成后，安装过程一直点击下一步就可以。

    安装完成后启动可能会报错提示：WSL2缺失。

    这里提供WSL2的下载地址：WSL2下载

    WSL2下载完成后，点击Docker对话框的Restart就安装完成了。

    创建映射文件

    在想要安装容器的盘内创建一个新的文件夹，并命名为Docker。

    在Docker文件夹下创建ql文件夹。

    在ql文件夹下依次创建如下文件夹。

    config、log、db、repo、raw、scripts

    这六个文件夹创建完成后，打开cmd窗口执行如下命令。（以D盘为例）

    D:\
cd Docker
docker run -dit -v D:/Docker/ql/config:/ql/config -v D:/Docker/ql/log:/ql/log -v D:/Docker/ql/db:/ql/db -v D:/Docker/ql/repo:/ql/repo -v D:/Docker/ql/raw:/ql/raw -v D:/Docker/ql/scripts:/ql/scripts -p 5700:5700 --name qinglong --hostname qinglong -e TZ=Asia/Shanghai --restart unless-stopped whyour/qinglong:latest

    执行命令后会自动下载镜像，并启动。

    在Docker程序中可以看到创建的镜像。

    启动镜像

    在Docker中可以停止、启动镜像。

    启动后访问http://localhost:5700 即可访问创建的服务。
